Basi di dati e linguaggio SQL

Concetti Base del Linguaggio SQL

SQL (Structured Query Language) è il linguaggio standard per la gestione dei database relazionali.

Cos'è SQL?

SQL è un linguaggio dichiarativo utilizzato per:

  • Creare e modificare strutture di database (DDL)
  • Manipolare dati (DML)
  • Interrogare database (DQL)
  • Controllare accessi e permessi (DCL)

Componenti Principali di SQL

1. Data Definition Language (DDL)

Comandi per definire la struttura del database:

CREATE TABLE Clienti (
    id INT PRIMARY KEY,
    nome VARCHAR(50),
    email VARCHAR(100)
);

ALTER TABLE Clienti ADD COLUMN telefono VARCHAR(15);

DROP TABLE Clienti;

2. Data Manipulation Language (DML)

Comandi per manipolare i dati:

INSERT INTO Clienti (id, nome, email)
VALUES (1, 'Mario Rossi', 'mario@example.com');

UPDATE Clienti
SET email = 'm.rossi@example.com'
WHERE id = 1;

DELETE FROM Clienti
WHERE id = 1;

3. Data Query Language (DQL)

Comandi per interrogare il database:

SELECT nome, email
FROM Clienti
WHERE nome LIKE 'Mario%'
ORDER BY nome;

SELECT c.nome, COUNT(o.id) AS num_ordini
FROM Clienti c
LEFT JOIN Ordini o ON c.id = o.cliente_id
GROUP BY c.nome
HAVING COUNT(o.id) > 5;

Clausole SQL Fondamentali

Clausola Descrizione Esempio
SELECT Specifica le colonne da recuperare SELECT nome, email

WHERE Filtra i record in base a condizioni WHERE età > 18
GROUP BY Raggruppa i record per valori GROUP BY città
HAVING Filtra i gruppi HAVING COUNT(*) > 10
ORDER BY Ordina i risultati ORDER BY cognome ASC
JOIN Combina dati da più tabelle JOIN Ordini ON Clienti.id = Ordini.cliente_id

Tipi di Dati Comuni in SQL

  • Numerici: INT, SMALLINT, DECIMAL(p,s), FLOAT
  • Testuali: CHAR(n), VARCHAR(n), TEXT
  • Data/Ora: DATE, TIME, DATETIME, TIMESTAMP
  • Booleani: BOOLEAN (TRUE/FALSE)
  • Binari: BLOB, BINARY
