Appunti e Studio

Questa sezione raccoglie i miei appunti, con esempi di codice e spiegazioni. Gli argomenti trattati sono suddivisi per linguaggi e strumenti.

HTML

HTML è il linguaggio fondamentale per costruire pagine web. Ecco un esempio di base:

                    
<!DOCTYPE html>
<html>
    <head>
        <title>Esempio HTML</title>
    </head>
    <body>
        <h1>Benvenuti nel mondo HTML!</h1>
    </body>
</html>

Python

Python è un linguaggio versatile, perfetto per scripting e sviluppo. Un esempio di struttura di controllo:

                    
# Controllo condizionale in Python
x = 10
if x > 5:
    print("x è maggiore di 5")
else:
    print("x è minore o uguale a 5")

SQL

SQL permette di interagire con i database. Un esempio di query per selezionare dati:

                    
-- Seleziona tutti i record da una tabella "utenti"
SELECT * FROM utenti WHERE età > 18;
